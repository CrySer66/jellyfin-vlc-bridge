## Fiche française

### Résumé

Prépare et lance les films, séries et collections Jellyfin dans VLC.

### Description

Jellyfin VLC Bridge ajoute une action « Lire avec VLC » directement dans l’interface Web de Jellyfin.

L’extension prépare le film, l’épisode, la série ou la collection affichée, puis demande à l’application Jellyfin VLC Bridge installée sur le même ordinateur de lancer VLC. Elle peut reprendre la lecture, préparer une liste de lecture et synchroniser la progression avec Jellyfin.

Fonctionnalités principales :

- bouton intégré à la barre d’actions Jellyfin ;
- lecture du fichier original avec VLC en HTTP Direct Play ou via un partage SMB configuré ;
- reprise à la dernière position connue ;
- préparation des épisodes suivants et des collections ;
- synchronisation de la progression avec Jellyfin ;
- choix de lecture facultatifs mémorisés localement ;
- aucune publicité, télémétrie ou collecte de données par le développeur.

L’application Windows Jellyfin VLC Bridge et VLC doivent être installés. La connexion au serveur Jellyfin se fait avec Quick Connect.

L’extension transmet uniquement les informations techniques nécessaires à l’application installée sur le même ordinateur. Aucune donnée n’est envoyée au développeur ou à un service publicitaire.

Projet indépendant, non affilié à Jellyfin, VideoLAN ou Google.

## English listing

Dans la fiche Chrome Web Store, ajoutez la langue **English**, puis utilisez les textes suivants.

### Summary

Prepare and launch Jellyfin movies, series and collections in VLC.

### Description

Jellyfin VLC Bridge adds a “Play with VLC” action directly to the Jellyfin Web interface.

The extension prepares the displayed movie, episode, series or collection, then asks the Jellyfin VLC Bridge application installed on the same computer to launch VLC. It can resume playback, prepare a playlist and synchronize progress with Jellyfin.

Main features:

- button integrated into the Jellyfin action bar;
- original-file playback with VLC through HTTP Direct Play or a configured SMB share;
- resume from the last known position;
- preparation of following episodes and collections;
- playback progress synchronization with Jellyfin;
- optional playback choices stored locally;
- no ads, telemetry or developer data collection.

The Jellyfin VLC Bridge Windows application and VLC must be installed. The Jellyfin server connection is configured through Quick Connect.

The extension only sends the technical information required by the companion application installed on the same computer. No data is sent to the developer or to an advertising service.

Independent project, not affiliated with Jellyfin, VideoLAN or Google.
